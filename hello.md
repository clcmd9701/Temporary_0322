# hello Git

## Git 명령어 요약

    clone: 원격 저장소를 로컬 컴퓨터에복사
    add: 스테이지 영역에 작업 파일 추가
    commint: 스테이지 영역의 파일들을 가지고 커밋을 만듦
    push: 원격 저장소에 커밋 업로드

### Branch 변경하기
    Branch: 기존 내용을 유지하고 새로운 내용을 추가할 때 사용
    CheckOut: 특정 Branch(Commit)로 이동하여 작업을 해야 할 때 사용
    소스트리에서 체크아웃 하는법: 브랜치 이름 더블클릭
